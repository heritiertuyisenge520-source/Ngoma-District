## Notes

1. **Indicators with sub-indicators** are marked with their parent indicator and show the detailed breakdown
2. **"-" values** indicate no target for that specific quarter
3. **All targets are fixed** and represent the expected achievements for each quarter
4. **Annual targets** are either the sum of quarterly targets or the final expected achievement
5. **Percentage indicators** show the expected percentage achievement for each period
6. **Numeric indicators** show the expected quantity or count for each period

This dataset provides the complete reference for all fixed targets used in the progress calculation system.
